Aprendendo CSS

Voltar para HTML

Sintaxe

seletor{propriedade: valor; }

Como Incluir

Inline

Estilo aplicado diretamente na tag HTML.
Ex.: style="color: red;"
Uso rápido, mas difícil de manter.

Interno

CSS dentro da tag style no head.
Vale para toda a página.
Melhor organização que o inline.

Externo

CSS em um arquivo .css separado, ligado com .
Reutilizável, organizado e a forma recomendada.

Seletores de elemento

p {...}

.minha-classe {...}

#meu-id {...}

Fontes

font-family - formatar a familia da fonte

font-size - mudar tamanho da fonte

font-style - quando tem italico ou não

font-weigth - se está em negrito ou normal

Text

text-aling - esquerda, direita, centralizar ou justificar

text-decoration - colocar ou tirar underline

text-ident - definir indetanção

text-transform - colocar texto em maiusculo ou minusculo
